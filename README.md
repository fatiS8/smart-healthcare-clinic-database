# Smart Healthcare Clinic Database
A database system project designed for a multi-clinic healthcare network. The system manages patients, doctors, appointments, prescriptions, inventory, billing, and user roles. The project demonstrates database design, normalization, data modeling, SQL query development, and healthcare data management concepts.

## Features
- Patient management
- Appointment scheduling
- Prescription management
- Inventory tracking
- Billing and invoicing
- Role-based access control
  
## Technologies Used
- Database design
- ER modeling
- Database normalization
- SQL
